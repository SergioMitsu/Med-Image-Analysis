# MedAnalyzer - Sistema de Análise de Imagens Médicas

Sistema web para análise automatizada de imagens médicas usando IA.

## Funcionalidades
- Login com 3 níveis: Admin, Médico, Paciente
- Upload de imagens (raio-X, ressonância, etc.)
- Análise automática com IA
- Histórico de análises
- Dashboard administrativo

## Como usar
1. Acesse: link
2. Cadastre-se como paciente
3. Faça upload de uma imagem médica
4. Veja o resultado da análise
